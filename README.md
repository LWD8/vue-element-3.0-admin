# 精简版截图
![image](https://github.com/LWD8/
vue-element-3.0-admin/blob/main/public/images/img1.png)

- Install dependencies
```
yarn install
```

- Compiles and hot-reloads for development
```
yarn run dev
```

- Compiles and minifies for production
```
yarn run build
```

- Lints and fixes files
```
yarn run lint
```
